# Job Market Analytics

A business analytics project exploring U.S. job market trends using real-world job posting data.

## Project Overview

This project analyzes over 72,000 Lightcast job postings to identify trends in Data Science and Business Analytics, including:

- In-demand skills
- AI & Machine Learning trends
- Industry hiring demand
- Salary analysis
- Text Mining
- K-Means Clustering
- Interactive Data Visualization
